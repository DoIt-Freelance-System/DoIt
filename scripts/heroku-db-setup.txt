INFO:
1. Free Heroku DB is Postgres.
2. Heroku Postgres DB launches in about a minute, so at first start be patient.
3. Heroku Postgres DB go in sleep mode after 30 minutes of idle state.
4. Heroku Postgres DB uses SSL, so you need to specialize it while connecting to it.

To Set up the connection in IDEA:
a) You need to specify credentials (You can find them upper)
b) it in IDEA in Setup DB properties in Tab 'Advanced' set:
   SSL=true
   sslfactory=org.postgresql.ssl.NonValidatingFactory
c) in database.properties in jdbc.url we need to specify it as well using line shown down:
   ?ssl=true&sslfactory=org.postgresql.ssl.NonValidatingFactory
